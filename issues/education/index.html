Education

A BROKEN SYSTEM

 

College Tuition: All public colleges and universities should be tuition free.

K-12 Eduction: No Child Left Behind should seriously be overhauled as it is currently ineffective and wasteful.

Common Core:The idea of Common Core Standards are well-intentioned, however the current implementation of the standards lack scientific foundation and ignore necessary changes in areas like early childhood education.

Charter Schools: All schools that receive tax dollars should be held the same standards and must be run in a way that ensures a beneficial outcome for all students.
